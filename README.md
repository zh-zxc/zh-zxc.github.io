# zh-zxc 的 GitHub Pages 个人网站

欢迎访问我的 GitHub Pages 个人网站！这是一个展示我个人项目、技能和联系方式的平台。

## 网站特性

- 响应式设计，适配各种设备
- 现代化的用户界面
- 展示个人技能和项目
- 包含 GitHub Pages 托管的项目链接

## 页面结构

- **首页**: 展示个人信息、技能和项目
- **关于我**: 个人介绍和背景
- **GitHub Pages 项目**: 展示使用 GitHub Pages 托管的项目
- **技能与技术栈**: 技术能力展示
- **联系方式**: 联系方式和社交媒体链接

## 如何使用 GitHub Pages

1. 将此代码推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择主分支作为源
4. 等待构建完成

## 自定义内容

要自定义此网站，请修改 `index.html` 文件中的以下部分：

- 个人信息和描述
- 项目链接和描述
- 联系方式
- 技能列表

## 技术栈

- HTML5
- CSS3
- 响应式设计

## 部署

此网站可以直接部署到 GitHub Pages，只需：

1. 创建一个名为 `<your-username>.github.io` 的仓库
2. 将此代码推送到主分支
3. 在 Settings → Pages 中启用 GitHub Pages
